# 项目运行说

#### 1、安装命令

```
npm install --legacy-peer-deps 
```

#### 2、注释事项

1. 在安装失败后，删除node_modules、package-lock.json

2. 必要时检查npm源地址，命令：

   ```
   npm config get registry
   ```

   应输出

   ```
   https://registry.npmjs.org
   ```

   如果不是官方地址就更换一下，然后再运行安装命令

   ```
   npm config set registry https://registry.npmjs.org
   ```
